Troubles that occurred During the Project 8


1. ROLLING WEEK BACK ACTUAL

•        There where no fiscal week. The   week back actual column is a rolling week from today until seven days back as of current week.

2. SOURCE DATA ISSUE

•        There was a lot of null values including the Ticket Id and Agent name.

3. SCATTER PLOT VISUALIZATION

•        To display the scatter plot bubbles, change dynamically we need Average line that divides the values in to four quadrants. The average line can only be fixed value so instead of that we use slicer.

4. KPI CALCULATION

•        There was no target value to show it as goal.

•        The KPI calculation was based on WKBK that doesn’t show a trend.
